Scene - Apfel against Mantana

Info: 

post-demo, animated/gameplay, final boss

Apfel tries to stop Mantana while a lot of flashbacks and flash-forwards appear in a countdown.

Mid scene the game window gets forcefully downscaled to 640x480, you will not be able to fullscreen or change the window size because it will alter its size while the animation is playing like what happens in (S) Cascade from Homestuck and (S) Game Over, the aspect ratio shouldn't exceed 16:9 because otherwise that would be a pain to port.

640x480 -> 640x360 -> 960x540

(Make an exception for 4K monitors, add a conditional that upscales the window by 2 depending on the monitor size.)

The music will be specifically composed so that in sections where you have to make a selection mid-scene it loops that section until you choose the option, then when the loop ends it continues.
