Paradox clock

Tags: #item
This clock is found inside Emilios journal, as you progress in the game the clock will count up to 12, the only thing is that there are no numbers, instead of numbers there are zodiac signs. The clock hits 12 once you get to the point where you need to go back in time. Emilio found it in the road where he had the accident because back-in-time-Apfel dropped it there, so the clock is a Paradox entity, it didnt come from anywhere. It has a hole shaped like the Octagram necklace, if you put it there youll unlock the time travel powers of the clock.

(Switch the Libra (12) sign with Capricorn, since Capricorn is linked to the planet Saturn, known to be associated with The ruler of time)
