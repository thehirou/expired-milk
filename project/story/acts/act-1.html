ACT 1

Tags: #story

This act goes from the beginning of the game to Blume waking up / Introduction to Mantana A.K.A. the end of the Demo.

Load game

_Game loads up, The Player is greeted by a fake load screen featuring the main symbol of the game, the Octagram.

Once it finishes loading, you'll be able to control a faceless character to move it around a room, this way we can teach The Player the controls without breaking the 4th wall. In that room you can go 2 different ways, either exit the game or create a new log.

If you go through the exit game way the game will close, if not, the story will begin. This kind of menu is not shown again but it's replaced with a more traditional one, less uncanny and featuring Apfel or whichever character you were controlling the moment you saved instead of the faceless character.
